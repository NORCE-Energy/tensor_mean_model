The code in this repository was used to generate results and figures for the paper Oliver, Dean S, A Tensor Mean Model for Reservoir Optimization Under Uncertainty, *Mathematical Geosciences*, 2026.


This Julia code computes a tensor effective model for permeability for use in computing the expected production from a reservoir with uncertainty in permeability. Although the method is fairly general, the code loads reservoir data for the Egg model and generates a tensor mean model for that reservoir model. In order to run the code, it will be necessary to download the JutulDarcy simulator and the data set for the Egg model.
